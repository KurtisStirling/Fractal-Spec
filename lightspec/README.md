Lightspec is a lightweight spec-driven development framework for use with Claude Code.

## Quick Start

1. Copy this folder to your project root
2. **Read `constitution/CONSTITUTION.md`** (framework structure and principles)
3. Edit `constitution/GOALS.md` (define what you're building)
4. Edit `constitution/RULES.md` (add project rules starting at R4)
5. Copy `specs/SPEC-TEMPLATE.md` to create your first spec
6. Read `.claude/CLAUDE.md` on every session start
